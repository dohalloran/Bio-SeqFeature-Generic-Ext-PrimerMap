##Installation
1. Download and extract the PrimerMap_Perl.zip file  
`tar -xzvf PrimerMap_Perl.zip`  
2. The extracted dir will be called PrimerMap_Perl  
  `cd PrimerMap_Perl`  
  `perl Makefile.PL`  
  `make`  
  `make test`  
  `make install`  

##Usage 
Run as follows:  
  `perl driver.pl`
